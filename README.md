# 🚚 Sistema de Gestión de Envíos

Presentamos un sistema de gestion de envios que permite a Operadores y Supervisores la creacion y seguimiento de envios, gestion de estados, busqueda y filtrado de envios, ademas de gestionar la informacion de los clientes.

---

## 📌 Funcionalidades

* Gestión de clientes
* Creación y seguimiento de envíos
* Gestión de estados (En Sucursal, En Transito, Entregado)
* Búsqueda y filtrado de envíos

## 🚀 Instalación

1. Clonar el repositorio:

```bash
git clone https://github.com/gonzalomartinez520/LogiTrack.git
```

2. Entrar al proyecto:

```bash
cd LogiTrack
```
(Depende que utilicemos en el proyecto, hara falta agregar mas pasos para que el usuario pueda utilizarlo)

3. Instalar dependencias:

```bash
npm install
```

4. Ejecutar el proyecto:

```bash
npm run dev
```

## 🌿 Estrategia de ramas

Se utiliza una estrategia basada en:

* `main`: código estable
* `develop`: integración de funcionalidades

### Ramas de trabajo:

Se crean desde `develop`:

* `feature/*`: nuevas funcionalidades
* `bugfix/*`: corrección de errores

---

## ✍️ Convención de commits

Formato:

```
tipo: descripción corta
```

Tipos:

* `feat`: nueva funcionalidad
* `fix`: corrección de errores
* `docs`: documentación
* `style`: cambios de formato
* `refactor`: mejoras del codigo
* `test`: pruebas

Ejemplos:

```
feat: agrega busqueda de envio
fix: corrige cambios de estados del envio
docs: actualiza README
```
